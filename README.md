# Labyrinthe-M1

## Build

`mvn clean package`

## Test

`mvn test`

## Run

Exécuter le .jar créé dans le dossier target avec un double clique ou bien en exécutant la commande: `java -jar .\target\labyrinthe-1.0.jar` depuis le dossier principal
